## AI安全

#### Masked-AI
- url: https://github.com/cado-security/masked-ai
- 说明

Masked-AI 是一个 Python SDK 和 CLI 包装器，可以更安全地使用公共 LLM（语言模型）API，例如 OpenAI/GPT4。它通过以下方式做到这一点：
（1）用虚假数据替换敏感数据（例如电子邮件地址）
（2）将请求发送到 API
（3）然后将敏感数据替换回输出中
结果是您从 API 获得相同的输出，而无需发送敏感数据。
